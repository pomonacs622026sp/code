# Package Structure Demo

```bash
# Compile
javac -d bin src/edu/helper/math/Calculator.java

# Run
java -cp bin edu.helper.math.Calculator
```

Things to try:

* Moving the files around
* Changing file names
* Changing directory names
* Changing class names
* Compiling without the `-d` flag
* Running without the `-cp` flag
* Compiling from the wrong directory
* Running from the wrong directory
* Running with extension
* Running from within `bin` (without the `-cp` flag)
